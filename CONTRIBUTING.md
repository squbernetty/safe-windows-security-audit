# Contributing

Contributions are welcome.

This project focuses on **safe, read-only Windows security auditing**.

Principles:
- No configuration changes
- No registry modifications
- No service manipulation
- No firewall rule changes

Scripts must remain **read-only inspection tools**.

If you want to contribute:

1. Fork the repository
2. Create a branch
3. Make your changes
4. Submit a Pull Request

Please include a short explanation of what the change does and why it is useful.
